# RocketEvolve

Rocket Evolve is a AI that spawns 500 rockets that learn/evolve how to get to 1 of 2-3 target(s).
### Inspired by 'Smart Rockets' by The Coding Train ([Link to video!](https://www.youtube.com/watch?v=bGz7mv2vD6g))
